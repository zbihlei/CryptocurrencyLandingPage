What I use at work
HTML5
HTML5 was created as a single markup language that could combine the syntactic norms of HTML and XHTML. It extends, improves, and streamlines document markup, and adds a single API for complex web applications. HTML5 has many new syntactic features.
CSS3
CSS is a style language that allows you to style almost any HTML tag on a page. By adding styles, you can create beautiful page designs. CSS3 differs from CSS only in version.
Java Script
JavaScript is a programming language that is used to write the frontend and backend parts of websites, as well as mobile applications. Often in texts and training materials, the name of the language is shortened to JS. This is a high-level programming language, that is, the code in it is understandable and well read.
jQuery
jQuery is a lightweight, "write less, do more", JavaScript library. The purpose of jQuery is to make it much easier to use JavaScript on your website. jQuery takes a lot of common tasks that require many lines of JavaScript code to accomplish, and wraps them into methods that you can call with a single line of code.
React
React is a popular Javascript programming language library that is used to create web user interfaces. It has certain templates, which greatly simplifies and speeds up the process of frontend development. In simple terms, a React developer.
Node.js
Node. js is used as a functional app builder. Thanks to him, one programming language is enough for developing web applications, instead of several different ones for the front-end and back-end. We can say that he this environment is based on the principle of "JavaScript for everything".
Redux
Redux is a tool for managing data state and UI in entity-heavy JavaScript applications. It is a JavaScript library. The name reads "Redax" and is made up of two words: reduce and flux.
